# Cluster

A browser-based word puzzle game built with HTML, CSS, and Phaser 3.
Group four words that share a hidden connection before you run out of lives.

## How to Play

- You are given 16 words on a board
- Select 4 words that share a hidden connection
- Submit your guess with the Check button
- You have 4 lives — wrong guesses cost a life
- Use Shuffle to reorganize the board
- Use Clue to reveal a category hint (costs 2 lives)
- Find all 4 groups to win

## Features

- 30 built-in puzzles
- Profile setup with name and Sanrio character icon
- Game history saved locally in your browser
- Dark and light mode toggle
- Background music and sound effects
- Live timer
- Puzzle picker to choose specific puzzles

## Tech Used

- [Phaser 3](https://phaser.io) — game rendering and animation engine
- HTML, CSS, vanilla JavaScript — no framework, no build tools
- localStorage — profile and history persistence
- Press Start 2P and Silkscreen — pixel fonts via Google Fonts
